# YouGood OS v1.0

這是「優夠好有限公司」的 Obsidian Vault
也是 Noah 經營人生風險管理顧問品牌的作業系統

請從 [[Dashboard]] 開始

## 核心入口
- [[Dashboard]]：整套系統首頁
- [[Brand DNA]]：品牌定位與核心主張
- [[人生三防線]]：核心 IP
- [[Business]]：商業模式與產品流程
- [[Customer Journey]]：客戶旅程
- [[GPT Prompt]]：AI 協作入口

## 系統架構
- [[Voice DNA]]：Noah 的文案語氣
- [[Visual DNA]]：視覺方向
- [[Product Ladder]]：產品階梯
- [[Content Strategy]]：內容策略
- [[成交流程]]：銷售流程
- [[客戶模板]]：CRM 標準格式

## 使用原則
這套 Vault 不是資料倉庫
而是 Noah 的品牌、產品、內容、銷售、CRM 與 AI 協作中樞

每次新增內容時
先想它應該回到哪一條主線：

1. 品牌是否更清楚
2. 產品是否更好成交
3. 客戶是否更容易被理解
4. Noah 是否更省力

[[Dashboard]] 是整套系統首頁
所有重要工作都從那裡進入
